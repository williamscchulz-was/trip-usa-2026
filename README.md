# Trip USA · Dec 2026 — Jan 2027

Roteiro visual da viagem em família Schulz · Horn · Pitz pelos Estados Unidos.

- **Data:** 20 dez 2026 → 06 jan 2027 (17 noites)
- **Grupo:** 6 adultos + 1 bebê (Louise, ~10 meses durante a viagem)
- **Rota:** NVT → Washington DC → New York → Boston → Park City (Utah) → Orlando → Miami → NVT

## Como abrir

Abre `index.html` no browser. É um single-page auto-contido com fontes externas do Google Fonts e imagens via loremflickr.

## Stack

- HTML + CSS puro (sem build step)
- DM Sans + JetBrains Mono (Google Fonts)
- Paleta azul minimal: `#26282B`, `#353941`, `#5F85DB`, `#90B8F8`
